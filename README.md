# 刷题

选择判断题练习工具。导入题库、组卷模考、错题重练，数据全部留在浏览器本地，不上传任何服务器。

## 它能做什么

- **导入题库**：粘贴文本，或拖入 `.txt` / `.docx` / `.pdf`。解析器兼容多种常见格式——题型标记 `[单选]/[多选]/[判断]`、各式题号、行内答案、以及结尾的 `1-5 ABDFA` 式统一答案块；无法识别的题目会列出原因而不是被悄悄丢掉。
- **多题库管理**：建立多个题库并随时切换、重命名、删除。
- **两种模式**：练习模式答完即时对错与解析；考试模式计时、交卷后统一判分。
- **组卷设置**：按题型与题量抽题，可打乱题目顺序与选项顺序——答案绑定选项内容，乱序后依然判分正确。
- **错题本**：答错自动收录，答对一次自动移出，可一键重练。
- **成绩与历史**：分数环、逐题复盘、历史趋势与平均分统计。
- **细节**：逐题翻页支持键盘流（方向键切题、数字键作答、`F` 标记），深浅色主题，移动端适配，刷新可续做未完成的练习。

## 技术栈

React 19 · TypeScript · Vite · Tailwind CSS v4 · lucide-react。题库解析为独立模块并有 Vitest 单测覆盖；状态持久化到 localStorage。

## 本地开发

```bash
bun install
bun dev          # 启动开发服务器
bun test         # 运行单元测试
bun run build    # 生产构建到 dist/
```

## 题库文本格式

最简形式：

```
[单选] 1. 中国的首都是哪个城市？
A. 上海
B. 北京
C. 广州
答案：B

[判断] 2. 地球是太阳系中最大的行星。
答案：错
```

答案也可以统一放在结尾：

```
1. 甲？
A. x  B. y

2. 乙？
A. m  B. n

参考答案
1-2 BA
```

## 隐私

题库与成绩只保存在你当前浏览器的本地存储里，不会发送到任何服务器。换设备或清理浏览器数据会丢失，可用「导出 JSON」备份题库。
